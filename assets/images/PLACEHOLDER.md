# Images Placeholder

Ce fichier sert de placeholder. Remplacez-le par vos vraies images.

## Comment Ajouter vos Images

1. **Supprimez ce fichier** (PLACEHOLDER.md)
2. **Ajoutez vos images** dans ce dossier
3. **Utilisez-les** dans vos fichiers Markdown avec :

```markdown
![Description]({{ '/assets/images/votre-image.png' | relative_url }})
```

## Exemples d'Images à Créer

Pour votre documentation médicale, vous pourriez avoir besoin de :

- **Schémas d'architecture** : Diagrammes montrant les composants du système
- **Diagrammes de flux** : Processus et workflows
- **Captures d'écran** : Interfaces utilisateur, résultats d'analyse
- **Graphiques** : Résultats de performance, comparaisons
- **Schémas médicaux** : Anatomie, processus biologiques (si pertinent)

## Outils pour Créer des Images

- **Draw.io** : https://app.diagrams.net/ (gratuit, en ligne)
- **Excalidraw** : https://excalidraw.com/ (gratuit, style dessin à la main)
- **Canva** : https://www.canva.com/ (gratuit avec compte)
- **Figma** : https://www.figma.com/ (gratuit pour usage personnel)

## Format Recommandé

- **Format** : PNG pour les schémas, SVG pour les diagrammes vectoriels
- **Taille** : 1200-1920px de largeur
- **Résolution** : 72-150 DPI (suffisant pour le web)
- **Poids** : < 500KB par image (optimisez avec TinyPNG)
